Pomodoro Clock

This is an Advanced Front End Developer project from the FreeCodeCamp curriculum. The user stories to be completed are:

  • I can start a 25 minute pomodoro, and the timer will go off once 25 minutes has elapsed.
  • I can reset the clock for my next pomodoro.
  • I can customize the length of each pomodoro.

This was made using: Flexbox, SCSS, and native JS. No CSS or JavaScript frameworks or libraries were used. The code can be seen on Github.
